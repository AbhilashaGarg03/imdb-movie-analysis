# IMDB Movie Database Analysis

## Project Overview

SQL-based analysis of IMDB movie database extracting insights on film performance, ratings, cast, and industry trends.

## Objective

Analyze movie data to:
- Identify top-performing films
- Analyze genre trends
- Explore cast and crew relationships
- Understand rating patterns

## Dataset

IMDB movie database including:
- Film metadata (title, year, budget, revenue)
- Ratings and reviews
- Cast and crew information
- Genre classifications

## Analysis Queries

SQL queries analyzing:
- Top-rated movies
- Box office performance
- Genre trends
- Actor/Director statistics
- Temporal trends
